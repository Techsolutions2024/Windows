## Getting Started

### Initial Setup

1. **Install PhotoBooth Pro**
   - Run the installer
   - Follow the installation wizard
   - Launch PhotoBooth Pro from desktop shortcut

2. **Connect Your Camera**
   - Connect Canon DSLR via USB cable
   - Turn on the camera
   - Set camera to "PC Remote" mode (if available)
   - Click "Connect Camera" in PhotoBooth Pro

3. **Configure Displays**
   - Connect second monitor for guest display (optional)
   - Go to Settings > Display
   - Select which monitor for operator and guest screens

### First Launch

When you first launch PhotoBooth Pro:

1. **Welcome Screen** appears
2. Click "Connect Camera" to detect your Canon camera
3. Camera information will display once connected
4. You're ready to start!

## Setting Up Your First Event

### Creating an Event Profile

1. **Open Settings**
   - Click "Settings" button in top menu
   - Go to "Events" tab

2. **Create New Event**
   - Click "New Event"
   - Enter event details:
     - Event Name (e.g., "Sarah & John's Wedding")
     - Event Type (Wedding, Birthday, Corporate, etc.)
     - Event Date
     - Venue

3. **Configure Photo Settings**
   - **Photo Count**: Choose 1-4 photos per session
   - **Countdown**: Set countdown duration (0-10 seconds)
   - **Interval**: Time between photos in multi-shot mode

4. **Select Template**
   - Browse template library
   - Select a template that matches your event theme
   - Preview how photos will look

5. **Enable Features**
   - ☑ Enable Printing (if you have a printer)
   - ☑ Enable Sharing (email, QR code)
   - ☑ Enable Green Screen (if using)
   - ☑ Collect Guest Emails

6. **Add Branding**
   - Upload event logo
   - Add watermark text
   - Customize colors

7. **Save Event Profile**
   - Click "Save"
   - Your event is now ready!

## Operating the Photobooth

### Starting a Session

1. **Select Event Profile**
   - Choose your event from dropdown
   - Settings will load automatically

2. **Start Live View**
   - Click "Start Live View"
   - Camera preview appears on screen

3. **Open Guest Display**
   - Click "Guest Display" button
   - Full-screen interface opens on second monitor
   - Shows "Touch to Start" attract screen

4. **Start Session**
   - Click "Start Session"
   - Photobooth is now active!

### Guest Experience

1. **Attract Screen**
   - Guest sees animated "Touch to Start" screen
   - Shows event name and branding

2. **Touch to Begin**
   - Guest touches screen anywhere
   - Countdown begins immediately

3. **Countdown**
   - Large countdown numbers (3-2-1)
   - Audio beeps for each count
   - "Smile!" message appears

4. **Photo Capture**
   - Camera captures photo
   - Flash fires (if enabled)
   - For multi-shot: countdown repeats

5. **Processing**
   - "Processing..." screen shows
   - Template overlay applied
   - Filters applied (if enabled)

6. **Preview Screen**
   - Guest sees final photo with template
   - Action buttons appear:
     - **Print** - Send to printer
     - **Share** - Email or QR code
     - **Retake** - Start over
     - **Done** - Finish session

7. **Sharing Options**
   - **QR Code**: Scan to download photo
   - **Email**: Enter email address to receive photo
   - Both options available simultaneously

8. **Thank You Screen**
   - "Thank You!" message
   - Returns to attract screen after 5 seconds

### Operator Controls

While guests are using the photobooth, you can:

- **Monitor Status**: Watch live view and session stats
- **Adjust Settings**: Change countdown, photo count, etc.
- **Switch Templates**: Change template between sessions
- **View Gallery**: See all photos taken
- **Check Printer**: Monitor paper and ink levels
- **End Session**: Stop photobooth when event ends

## Creating Custom Templates

### Template Designer

1. **Open Template Editor**
   - Go to Settings > Templates
   - Click "New Template"

2. **Set Template Size**
   - Choose print size (4x6, 5x7, etc.)
   - Or custom dimensions
   - Resolution: 300 DPI recommended

3. **Add Background**
   - Click "Add Background"
   - Upload background image
   - Or choose solid color

4. **Add Photo Zones**
   - Click "Add Photo Zone"
   - Drag to position
   - Resize as needed
   - Set border style (optional)
   - Repeat for multiple photos

5. **Add Text**
   - Click "Add Text"
   - Enter text (event name, date, etc.)
   - Choose font, size, color
   - Position on template

6. **Add Graphics**
   - Click "Add Graphic"
   - Upload logo or decoration
   - Position and resize
   - Adjust opacity

7. **Preview**
   - Click "Preview"
   - See how template looks with sample photos
   - Make adjustments

8. **Save Template**
   - Give template a name
   - Add description
   - Save to library

### Template Tips

- **Photo Zones**: Leave space between zones for visual separation
- **Text**: Use high-contrast colors for readability
- **Logos**: PNG format with transparency works best
- **Resolution**: Use high-resolution images (300 DPI)
- **Testing**: Test print before event to verify layout

## Printer Setup

### Connecting Printer

1. **Install Printer Drivers**
   - Install manufacturer's drivers
   - Connect printer via USB or network
   - Ensure printer appears in Windows

2. **Configure in PhotoBooth Pro**
   - Go to Settings > Printer
   - Click "Detect Printers"
   - Select your printer from list

3. **Set Paper Size**
   - Choose paper size (4x6, 5x7, etc.)
   - Must match paper loaded in printer

4. **Set Print Quality**
   - Draft: Fast, lower quality
   - Normal: Balanced
   - High: Best quality (recommended)

5. **Test Print**
   - Click "Test Print"
   - Verify output quality
   - Adjust settings if needed

### Printer Settings

- **Auto-Print**: Automatically print after each session
- **Copies**: Number of copies per session (1-4)
- **Print Queue**: View and manage print jobs

### Supported Printers

- **Canon SELPHY** (CP1300, CP1500, etc.)
- **DNP DS** series (DS620A, DS820A, etc.)
- **Mitsubishi CP** series (CP-D90DW, etc.)
- **HiTi P** series (P525L, P750L, etc.)
- Any Windows-compatible photo printer

## Troubleshooting

### Camera Issues

**Camera Not Detected**
- Ensure camera is powered on
- Check USB cable connection
- Try different USB port
- Close other camera software (EOS Utility)
- Restart PhotoBooth Pro

**Live View Not Working**
- Set camera to PC Remote mode
- Check camera mode dial (use P, Av, Tv, or M)
- Disable camera auto-power off
- Update camera firmware

**Photos Not Downloading**
- Check camera memory card
- Ensure "Save To" is set to "Host" or "Both"
- Check available disk space
- Verify download folder permissions

### Printer Issues

**Printer Not Found**
- Check printer power and connection
- Install/update printer drivers
- Restart printer
- Restart PhotoBooth Pro

**Print Quality Poor**
- Check ink/ribbon levels
- Clean print head
- Use correct paper type
- Increase print quality setting

**Prints Not Starting**
- Check print queue
- Clear stuck jobs
- Verify paper is loaded correctly
- Check for paper jams

### Display Issues

**Guest Display Not Showing**
- Check second monitor connection
- Verify display settings in Windows
- Extend displays (don't duplicate)
- Restart PhotoBooth Pro

**Touch Screen Not Responding**
- Calibrate touch screen in Windows
- Check touch driver installation
- Test touch in other applications
- Restart computer

### Performance Issues

**Slow Live View**
- Reduce live view resolution
- Close other applications
- Check CPU usage
- Update graphics drivers

**Photos Processing Slowly**
- Simplify template (fewer elements)
- Reduce image resolution
- Disable real-time filters
- Upgrade computer hardware

## Tips & Best Practices

### Before the Event

1. **Test Everything**
   - Test camera connection
   - Test live view
   - Test photo capture
   - Test printing
   - Test sharing features

2. **Prepare Backup**
   - Bring spare camera battery
   - Bring spare USB cable
   - Have backup printer paper
   - Have backup ink/ribbon

3. **Setup Early**
   - Arrive 1-2 hours before event
   - Set up in designated area
   - Test lighting
   - Do test captures

4. **Lighting**
   - Use external flash or softbox
   - Avoid harsh overhead lighting
   - Test exposure settings
   - Adjust ISO if needed

### During the Event

1. **Monitor Regularly**
   - Check camera battery
   - Check printer paper/ink
   - Check disk space
   - Watch for errors

2. **Assist Guests**
   - Explain how to use photobooth
   - Help with posing
   - Encourage props
   - Ensure good lighting

3. **Backup Photos**
   - Enable auto-backup
   - Periodically copy to external drive
   - Use cloud backup if available

### After the Event

1. **Export Photos**
   - Go to Gallery
   - Select all photos
   - Export to folder
   - Organize by event

2. **Generate Report**
   - View session statistics
   - Export analytics
   - Share with client

3. **Backup Everything**
   - Copy all photos to external drive
   - Backup event configuration
   - Save templates used

4. **Maintenance**
   - Clean camera sensor
   - Clean printer
   - Update software
   - Review logs for errors

### Green Screen Tips

1. **Lighting**
   - Even lighting on green screen
   - Separate subject from background
   - Avoid shadows on green screen

2. **Distance**
   - Subject 3-6 feet from green screen
   - Prevents green spill on subject

3. **Clothing**
   - Avoid green clothing
   - Avoid reflective materials
   - Solid colors work best

4. **Settings**
   - Adjust tolerance for clean key
   - Use edge feathering
   - Test with different backgrounds

### Template Design Tips

1. **Keep It Simple**
   - Don't overcrowd template
   - Leave breathing room
   - Focus on photos

2. **Brand Consistency**
   - Use event colors
   - Include logo
   - Match event theme

3. **Readability**
   - High contrast text
   - Large enough font size
   - Clear, legible fonts

4. **Print Considerations**
   - Design for print size
   - Use 300 DPI images
   - Test print before event
   - Account for printer margins

### Guest Experience Tips

1. **Clear Instructions**
   - Large, clear text
   - Simple steps
   - Visual cues

2. **Fast Response**
   - Minimize countdown
   - Quick processing
   - Instant preview

3. **Easy Sharing**
   - Large QR code
   - Simple email input
   - Multiple sharing options

4. **Props**
   - Provide fun props
   - Organize prop table
   - Encourage creativity
