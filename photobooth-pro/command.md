# Canon MultiCamCui Command List

Dưới đây là danh sách các mã lệnh được sử dụng trong Sample App `MultiCamCui` để điều khiển Camera Canon qua SDK.

| ID  | Command Name | Description |
| --- | :--- | :--- |
| **01** | Set Save To | Thiết lập nơi lưu ảnh (Camera, Host PC, hoặc Cả hai) |
| **02** | Set Image Quality | Chọn chất lượng ảnh (RAW, JPEG Large/Fine...) |
| **03** | Take Picture and download | Chụp ảnh ngay lập tức và tải về máy tính |
| **04** | Press Halfway | Nhấn nửa cò (lấy nét) |
| **05** | Press Completely | Nhấn chụp hoàn toàn |
| **06** | Press Off | Nhả nút chụp |
| **07** | TV | Chỉnh tốc độ màn trập (Shutter Speed) |
| **08** | AV | Chỉnh khẩu độ (Aperture) |
| **09** | ISO | Chỉnh độ nhạy sáng ISO |
| **10** | White Balance | Cân bằng trắng (Auto, Daylight, Cloudy...) |
| **11** | Drive Mode | Chế độ chụp (Single, Continuous, Timer...) |
| **12** | Exposure Compensation | Bù trừ sáng (EV) |
| **13** | AE Mode (read only) | Xem chế độ phơi sáng hiện tại (M, Av, Tv, P...) |
| **14** | AF Mode (read only) | Xem chế độ lấy nét (One-Shot, AI Servo...) |
| **15** | Aspect setting (read only) | Xem tỷ lệ khung hình |
| **16** | Get Available shots (read only) | Xem số lượng ảnh còn có thể chụp |
| **17** | Get Battery Level (read only) | Xem dung lượng pin |
| **18** | Edit Copyright | Chỉnh sửa thông tin bản quyền trong Exif |
| **20** | Get Live View Image | Lấy **một** frame hình Live View hiện tại |
| **21** | Start Live View | Bắt đầu chế độ Live View (mở màn trập, stream ảnh) |
| **22** | Stop Live View | Tắt chế độ Live View |
| **30** | All File Download | Tải tất cả ảnh từ thẻ nhớ về máy tính |
| **31** | Format Volume | Format thẻ nhớ |
| **32** | Set Meta Data(EXIF) | Ghi metadata EXIF vào toàn bộ ảnh |
| **33** | Set Meta Data(XMP) | Ghi metadata XMP vào toàn bộ ảnh |
| **34** | Current Storage | Kiểm tra thiết bị lưu trữ hiện tại |
| **35** | Current Folder | Kiểm tra thư mục hiện tại trên thẻ |
| **36** | Filename Setting | Cài đặt cách đặt tên file |
| **37** | Create Still Folder Setting | Tạo thư mục mới để lưu ảnh tĩnh |
| **40** | Rec mode on | Chuyển sang chế độ quay phim |
| **41** | Rec Start | Bắt đầu quay |
| **42** | Rec Stop | Dừng quay |
| **43** | Rec mode off | Quay lại chế độ chụp ảnh tĩnh |
| **44** | MovieRecVolume | Chỉnh âm lượng thu âm khi quay |
| **45** | MovieParamEx | Các tham số quay phim mở rộng |
| **50** | Select Picture Style | Chọn kiểu màu (Standard, Portrait, Landscape...) |
| **51** | Picture Style edit | Chỉnh sửa chi tiết Picture Style |
| **60** | Get Lens Name | Lấy tên ống kính |
| **61** | Lens IS Setting | Bật/Tắt chống rung ống kính |
| **62** | Aperture Lock Setting | Khóa khẩu độ |
| **63** | Focus Shift Setting | Cài đặt Focus Bracketing |
| **64** | [FPM] Drive Focus To Edge | Di chuyển tiêu cự ra vô cực hoặc cực cận |
| **65** | [FPM] Register Edge Focus Position | Lưu vị trí tiêu cự biên |
| **66** | [FPM] Get Current Focus Position | Lấy vị trí tiêu cự hiện tại |
| **67** | [FPM] Moving the Focus Position | Dịch chuyển tiêu cự thủ công |
| **70** | Auto power off | Cài đặt thời gian tự tắt máy |
| **71** | Screen Off Time | Cài đặt thời gian tắt màn hình |
| **72** | Viewfinder Off Time | Cài đặt thời gian tắt ống ngắm |
| **73** | Screen dimmer Time | Cài đặt thời gian làm tối màn hình |
| **80** | Digital Zoom Setting | Zoom kỹ thuật số |
| **81** | Color Filter (V10 only) | Bộ lọc màu (chỉ dòng V10) |
| **82** | AfLock State (V10 only) | Trạng thái khóa AF (chỉ dòng V10) |
| **83** | Brightness Setting (V10 only) | Độ sáng (chỉ dòng V10) |
| **84** | Power Zoom Speed | Tốc độ Zoom điện tử |
| **85** | Power Zoom Tele | Zoom vào (Tele) |
| **86** | Power Zoom Wide | Zoom ra (Wide) |
| **87** | Power Zoom Stop | Dừng Zoom |
| **88** | Power Zoom Position | Lấy vị trí Zoom |
| **90** | Target Flash Setting | Cài đặt đèn Flash |
| **91** | Flash Firing | Kích nổ Flash |
| **98** | IBIS HighResoShot | Chụp độ phân giải cao dùng chống rung (EOS R5) |
| **99** | WhiteBalanceShift | Dịch chuyển cân bằng trắng (WB Shift) |
| **100**| Take Picture Interval | Chụp ảnh theo khoảng thời gian |
